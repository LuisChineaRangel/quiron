quiron module

Created on Tue Dec 20 2022

@author: iluzioDev

This script implements Hand Recognition using Mediapipe and OpenCV. Includes features like:

  • Hand Recognition

  • Hand Movement Detection

  • Hand Gesture Recognition

  • Finger Counting

quiron.count_fingers(landmark, label)

Counts the number of fingers up in a hand

Parameters:

  • landmark – Landmark of one hand

  • label – Label of the hand (Left or Right)

Returns:

Number of fingers up

Return type:

int

quiron.detect_gestures(landmark, label)

Detects the gesture of a hand

Parameters:

  • landmark – Landmark of one hand

  • label – Label of the hand (Left or Right)

Returns:

The gesture if it is detected, False otherwise

Return type:

str or bool

quiron.main()

Main function of the program. Starts the WebCam Capture and uses the features implemented

Returns:

None

quiron.moving(landmark)

Detects if the hand is moving or not calculating the distance in a margin between the positions of 2 instances through time of the hand

Parameters:

landmark – Landmark of one hand

Returns:

True if the hand is moving, False otherwise

Return type:

bool
